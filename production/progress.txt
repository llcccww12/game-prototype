## 2026-04-27 - pc-001
- Implemented Virtual Joystick Movement
- Created virtual_joystick.gd with touch-based joystick
- Updated player.gd to use Global.joystick_input for movement
- Movement speed: 200px/s, dead zone: 10px, max drag: 50px
- Player sprite flips based on horizontal movement direction
- **Learnings:** Godot 4.6 uses CharacterBody2D for top-down movement, not KinematicBody2D (deprecated). Touch input uses InputEventScreenTouch and InputEventScreenDrag.

## 2026-04-27 - pc-002
- Implemented Game State Machine
- States: MENU, PLAYING, PAUSED, DEAD, VICTORY
- MENU -> PLAYING: main_menu "Start" button
- PLAYING -> DEAD: player HP reaches 0
- PLAYING -> VICTORY: boss HP reaches 0
- DEAD/VICTORY -> MENU: retry/menu buttons
- State transitions emit signals for other systems to react
- **Learnings:** Use enum for game states, emit signals from state changes so UI and gameplay can react.
